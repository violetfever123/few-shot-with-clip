# few-shot-with-clip

在这里可以存放大家看过的论文以及代码实验哦，同时也欢迎大家把阅读过的文献用 markdown 文件做成笔记哦 📖

> A shared space for research papers, code experiments, and reading notes on few-shot learning with CLIP.

---

## 📁 目录结构 / Repository Structure

```
few-shot-with-clip/
├── papers/          # 论文 PDF 或论文链接 / Research paper PDFs or links
├── notes/           # 论文阅读笔记 (Markdown) / Paper reading notes in Markdown
├── code/            # 代码实验 / Code experiments
└── README.md
```

---

## 📝 如何贡献 / How to Contribute

### 添加论文笔记 / Adding Paper Notes

1. 在 `notes/` 目录下新建一个 Markdown 文件，命名格式如下：
   ```
   notes/YYYY-MM_论文简称.md
   ```
   例如：`notes/2023-01_CLIP-few-shot.md`

2. 可以参考 [notes/TEMPLATE.md](notes/TEMPLATE.md) 模板来撰写笔记。

### 添加代码实验 / Adding Code Experiments

1. 在 `code/` 目录下新建一个子目录，以实验名称命名：
   ```
   code/实验名称/
   ```
2. 在子目录中添加代码文件和一个 `README.md` 说明实验内容。

### 添加论文 / Adding Papers

1. 可以在 `papers/` 目录下存放论文 PDF 文件，或者在 `papers/README.md` 中添加论文链接。
