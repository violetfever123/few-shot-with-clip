# 代码实验 / Code Experiments

在此目录下存放各类代码实验。每个实验放在独立的子目录中，并附有 README.md 说明。

---

## 目录结构建议 / Suggested Structure

```
code/
├── experiment-name/
│   ├── README.md       # 实验说明
│   ├── requirements.txt  # 依赖包
│   └── *.py / *.ipynb  # 代码文件
└── ...
```

## README 模板 / README Template

每个实验的 README.md 建议包含以下内容：

- **实验目的 / Purpose**: 本实验要验证什么？
- **数据集 / Dataset**: 使用的数据集
- **运行方法 / How to Run**:
  ```bash
  pip install -r requirements.txt
  python main.py
  ```
- **实验结果 / Results**: 关键指标和结论
